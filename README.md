Managed Code combines product design, software engineering and generative AI to turn complex ideas into working products. ## Current products

### AIBase

AIBase connects company knowledge, internal tools and operational workflows into enterprise AI agents. The platform supports cloud and on premises deployment, integrations across collaboration tools, strong privacy controls and support for major model providers.

### MCAF

MCAF keeps code, tests, documentation and `AGENTS.md` in one repository and turns AI assisted development into a repeatable workflow with clear rules and verification gates.

### GraphRAG for .NET

GraphRAG for .NET ports Microsoft's GraphRAG reference architecture to the modern .NET stack, with end to end indexing workflows, semantic deduplication, community detection and graph backends for Neo4j, PostgreSQL with Apache AGE and Azure Cosmos DB.

### SiloLinker

SiloLinker is a distributed link platform built on Orleans. It started as a scalable URL management system and evolved toward gated access, temporary links and creator monetization flows.

### Keyload

Keyload covers monitoring, incident detection and load testing for websites, APIs and infrastructure, with scenario based checks and a real time operations view.

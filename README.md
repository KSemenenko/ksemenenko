# Current Focus

## AIBase

Enterprise knowledge system powered by **GraphRAG and AI agents**.

Goal: give companies a working **AI memory layer** built on top of their internal knowledge.

Key ideas:

- knowledge graphs
- RAG pipelines
- agent orchestration
- internal company tools
- self-hosted AI

---

## MCAF

**Managed Code AI Framework**

Framework for building software together with AI coding agents.

Principles:

- tests as hard gates
- repository memory (`AGENTS.md`)
- deterministic builds
- documentation-first development

Goal: treat AI agents like **team members**, not prompt toys.

---

## GraphRAG for .NET

Full **.NET implementation of Microsoft GraphRAG**.

Features:

- document ingestion pipelines
- graph extraction
- semantic deduplication
- community detection
- multiple graph databases

Backends supported:

- Neo4j
- PostgreSQL / Apache AGE
- Azure CosmosDB

---

## SiloLinker

Distributed link infrastructure built with **Microsoft Orleans**.

Use cases:

- gated content
- creator monetization
- temporary links
- scalable URL routing

---

## Keyload

Monitoring and load testing platform.

Capabilities:

- website monitoring
- REST API monitoring
- infrastructure checks
- synthetic load testing
- incident detection
